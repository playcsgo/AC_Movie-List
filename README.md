# Movie List
A dumy-project of web application with a dumy-API url

## Features
- listing movies by AJAS and DOM
- searching moives by comparing string
- add movie to favorite by localStorage
- pagination

### Searching
type the key words in search input bar

### Favorite List
In index page, press the `+` button to add this movie to Favorite. You can check the collection by click Favorite in Nav Bar to Favorite page. You can remove that movie by clicking the `x` button in this page.
